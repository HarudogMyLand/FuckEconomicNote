# 绪论与金融体系

## 金融的内涵

- 金融是货币与信用的融合体，始于货币信用相关的所有活动的集合
- 金融学是研究各类经济主体金融活动和金融体系各部分相互关系与整体运行规律的科学。

### 金融体系

金融体系是指各种要素在一定制度法规框架下组成的系统性整体，主要包括

- 金融工具机器价格体系
- 金融机构体系
- 金融市场体系
- 金融监管体系
- 政策调控体系
- 金融基础设施服务体系

### 本质与功能

金融本质是服务，社会主义市场经济体制下的金融和信誉关键是为实体经济服务。

金融的本质是通过金融功能来实现的，金融功能通过金融活动发挥出来。

1. 提供货币充当支付中介
2. 满足投融资充当信用中介
3. 资源配置充当相关管理者
   1. 信息管理者
   2. 风险管理者
   3. 资产管理者

### 地位与作用

1. 金融是国民经济的血脉
2. 金融在现代经济中发挥核心作用
3. 金融是“国之大者”
   1. 核心竞争力
   2. 现代化建设
   3. 金融安全是国家安全重要组成

### 金融强国要求

- 六个强大：“强大的货币，强大的央行，强大的金融机构，强大的国际金融中心，强大的金融监管，强大的金融人才队伍。”
- 五大文章：科技金融、绿色金融、普惠金融、养老金融和数字金融
