# 中央银行制度与业务

中央银行是发行的银行，银行的银行，政府的银行。

## 中央银行

中央银行是**专门指定和实施货币政策、统一管理金融活动并代表政府协调对外金融关系**的金融管理机构。

央行是一国最重要的金融管理当局和宏观经济调控部门。

中国人民银行成立于1948年12月1日，由华北银行、北海银行、西北农民银行组建而成，负责人民币的发行和管理。

### 中央银行类型

- 单一制：一个国家或地区建立单独中央银行机构，使之全面行使银行职能；
- 跨国制：若干国家联合组建一家央行、行使全部或部分央行职能；
- 复合制：一家集央行与商行职能议题的大银行兼央行
- 准央行制：授权某个或几个商业银行部分行使央行权力

在中国，由“大统一”时代的复合制走向单一制垂直管理

中国人民银行的权利结构：

- 决策权：行长负责制
- 执行权：独立执行货币政策，履行职责，开展业务
- 监督权：人民银行对金融市场和银行业金融机构有一定的监督管理权

### 央行的性质与职能

特殊的银行、特殊的政府机构

**中央银行既是服务和引领金融体系发展的特殊银行，又是国家管理货币和金融的特殊政府机构。**

**发行的银行、银行的银行和政府的银行是中央银行的三大职能。**

集中与垄断货币发行权，向社会提供经济活动所需要的货币数量，并保证货币流通正常运行，维护币值稳定。

为商业银行及其他金融机构提供金融服务和支付保证，并监督管理各金融机构与金融市场业务活动的职能。

由政府授权对金融业实施监督管理、对宏观经济进行调控，代表政府参与国际金融事务，并为政府提供融资、国库管理等服务。

### 业务活动

业务活动遵循**非盈利性、流动性、主动性和公开性**。

资产业务、负债业务、其他业务

#### 资产业务：发钱的业务

央行把钱投出去、持有某些资产

##### 再贴现和再贷款业务

商业银行遇到资金短缺时，可把手里的优质票据（再贴现）或资产抵押（再贷款）给央行，换得资金。央行相当于 “银行的银行”，是最后贷款人。

##### 证券类业务

央行在金融市场上买卖政府债券（中国主要是国债），相当于 “直接向市场放钱或收钱”。公开市场操作，调节货币供应量
